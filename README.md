## 📖 Descrizione

Quiz interattivo a risposta multipla per prepararsi all'esame di **Informatica** del corso di Psicobiologia. Copre tutti gli argomenti principali del programma con spiegazioni dettagliate dopo ogni risposta.

Funziona direttamente nel browser, **senza installazioni** e **senza connessione internet** — perfetto da studiare su iPhone, iPad o PC.

## ✨ Funzionalità

| | |
|---|---|
| ✅ **Risposta singola e multipla** | Domande con una o più risposte corrette |
| 🔀 **Ordine casuale** | Domande e opzioni rimescolate ad ogni sessione |
| 💡 **Spiegazione dettagliata** | Feedback immediato dopo ogni risposta |
| 📋 **Recap finale** | Riepilogo con tutti gli errori, risposte corrette e spiegazioni |
| 📊 **Dashboard storico** | Storico degli ultimi 30 tentativi con punteggi e analisi errori |
| 💾 **Dati persistenti** | Risultati salvati in locale (localStorage), nessun server |
| 📱 **Mobile-first** | Tasti grandi, nessun zoom indesiderato, aggiungibile alla schermata Home |
| ⚡ **Zero dipendenze** | Un solo file HTML, nessuna libreria esterna |
| 🔁 **Tentativi illimitati** | Ripeti il quiz quante volte vuoi |

## 🚀 Come usarlo

### Sul telefono (consigliato)
1. Apri il link della **demo live** qui sopra con **Safari** (iPhone) o **Chrome** (Android)
2. Tocca *Condividi → Aggiungi a schermata Home* per usarlo come un'app

### In locale
1. Clona il repository:
   ```bash
   git clone https://github.com/T-Ivan2/Flashcard-quiz-Esame_Informatica_Facolta_Psicobiologia_L-24.git
   ```
2. Apri `flash-card-quiz.html` nel browser — nessun server necessario

## 🗺️ Flusso dell'app

```
Schermata iniziale
    ├── 📋 Inizia in ordine
    └── 🔀 Ordine casuale
            │
            ▼
    Quiz (domanda per domanda)
        → verifica risposta → spiegazione → prossima
            │
            ▼
    📋 Recap finale
        → errori con risposta corretta + spiegazione
            │
            ├── 🔄 Ricomincia
            └── 📊 Dashboard storico
                    → ultimi 30 tentativi espandibili
```
