## 🚀 Funzionalità

### Quiz
- **Ordine normale** — domande nella sequenza originale
- **Ordine casuale** — domande mescolate ad ogni avvio
- **Opzioni randomizzate** — le risposte cambiano posizione ad ogni sessione
- Supporto a risposte **singole** e **multiple**
- Feedback immediato dopo ogni risposta con spiegazione dettagliata

### Recap finale
Al termine di ogni sessione viene mostrata una schermata di riepilogo con:
- Punteggio e percentuale
- Tutte le risposte sbagliate con la risposta corretta evidenziata in verde e la tua in rosso
- Spiegazione per ogni errore

### Dashboard storico
Accessibile dopo il primo tentativo completato:
- Numero di tentativi, miglior risultato e media
- Lista degli ultimi 30 tentativi con data, ora e punteggio
- Ogni tentativo è espandibile per rivedere gli errori e le spiegazioni
- Pulsante per cancellare lo storico

---

## 💾 Salvataggio dati

I risultati vengono salvati nel **localStorage** del browser — persistono tra sessioni finché non si svuota la cache del browser o si cancella lo storico dalla dashboard.  
Nessun dato viene inviato a server esterni.
